## Pull Shark

> PR을 **2회 이상** 생성하고 `merge`하면 획득

### 방법

1. **새 Repository 생성**
   - `Public`으로 설정
   - `Add a README file` 체크

2. **새 브랜치 생성**
   - `Main` → `View all branches` → `New branch`
   - 예: `shark` 브랜치 생성

3. **브랜치에서 파일 수정 후 커밋**
   - `shark` 브랜치에서 `README.md` 수정
   - `Commit changes`로 변경사항 저장

4. **Pull Request 생성**
   - `Pull requests` → `New pull request`
   - `compare` 브랜치를 `shark`로 변경
   - `Create pull request` 클릭

5. **Merge**
   - `Merge pull request` 클릭해서 `main`에 병합

> 이 과정을 **2회 이상** 반복하면 뱃지 획득!

## YOLO

> Reviewer가 있는 PR을 리뷰 없이 `merge`하면 획득

### Pull Shark와의 차이점

Pull Shark와 동일한 과정이지만, **Reviewer를 추가**해야 합니다.

### 방법

1. **Collaborator 추가**
   - Repository `Settings` → `Collaborators` → `Add people`
   - 친구 또는 부계정을 등록

2. **브랜치에서 파일 수정 후 PR 생성**
   - Pull Shark와 동일하게 진행

3. **Reviewer 지정 후 Merge**
   - PR 생성 전, 우측 `Reviewers`에 추가한 collaborator 지정
   - `Create pull request` 후 바로 `Merge pull request`

> 리뷰 없이 merge하면 뱃지 획득!

## Galaxy Brain

> Discussion의 Q&A에서 답변이 **2회 이상** `Answer`로 채택되면 획득

### 방법

1. **Discussion 활성화**
   - Repository `Settings` → `Features` → `Discussions` 체크

2. **Q&A Discussion 생성**
   - `Discussions` 탭 → `New Discussion`
   - 카테고리를 반드시 **Q&A**로 선택
   - 친구 또는 부계정으로 질문 작성

3. **뱃지 받을 계정으로 답변 작성**
   - 메인 계정으로 로그인해서 답변 달기

4. **질문한 계정에서 채택**
   - 질문을 작성한 계정으로 다시 로그인
   - 달린 답변에서 `Mark as Answer` 클릭

> 이 과정을 **2회 이상** 반복하면 뱃지 획득!

| 뱃지 | 조건 | 핵심 포인트 |
|------|------|-------------|
